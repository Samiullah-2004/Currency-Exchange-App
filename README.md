# 💱 Currency Exchange App

A clean, responsive currency exchange web app that converts between world currencies using live exchange rates.

## 🌐 Live Demo
[Click here to view](https://samiullah-2004.github.io/Currency-Exchange-App/)

## ✨ Features
- Real-time exchange rates via API
- Supports multiple world currencies with flags
- Clean dark UI with smooth animations
- Fully responsive on mobile and desktop

## 🛠️ Built With
- HTML
- CSS (animations, dark theme)
- JavaScript (API integration)
- flagsapi.com
- cdn.moneyconvert.net
